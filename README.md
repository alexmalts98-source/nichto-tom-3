# Nothing Is Forever — Tom 3 Repository
*«Ничто не вечно» — Том 3 (приквел). Только материалы Тома 3.*

## Структура

```
/bible/
  world_map.md    — политическая карта + мелкие локации Тома 3
  timeline.md     — хронология Тома 3

/chapters/        — главы Тома 3 (добавлять по мере написания)

/critique/
  issues_log.md   — найденные противоречия

critic_prompt.md  — системный промпт для сессий с Клодом
```

## Протокол сессии

1. Вставить `critic_prompt.md`
2. Вставить `bible/world_map.md` + `bible/timeline.md`
3. Вставить главы для анализа (2–5 за раз)
4. Запросить: *«Проверь на все типы нарушений»*
5. Найденные проблемы → `critique/issues_log.md`
